# ShroudbAuth Go Client

Go HTTP client for the [ShroudbAuth](https://github.com/shroudb/shroudb-auth) Authentication service.

## Install

```bash
go get github.com/shroudb/shroudb-auth-go
```

## Quick Start

```go
package main

import (
    "context"
    "fmt"
    "log"

    shroudb_auth "github.com/shroudb/shroudb-auth-go"
)

func main() {
    client := shroudb_auth.NewClient(
        "http://localhost:4001",
        shroudb_auth.WithKeyspace("my-keyspace"),
    )

    ctx := context.Background()

    // Sign up a user
    result, err := client.Signup(ctx, "alice", "s3cret", nil)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println("Access token:", result.AccessToken)

    // Check the session
    session, err := client.Session(ctx)
    if err != nil {
        if shroudb_auth.IsUnauthorized(err) {
            log.Fatal("token expired — call Refresh()")
        }
        log.Fatal(err)
    }
    fmt.Println("User:", session.Claims)
}
```

## Client Options

```go
client := shroudb_auth.NewClient("http://localhost:4001",
    shroudb_auth.WithKeyspace("my-app"),
    shroudb_auth.WithAccessToken("existing-jwt"),
    shroudb_auth.WithRefreshToken("existing-rt"),
    shroudb_auth.WithHTTPClient(&http.Client{Timeout: 5 * time.Second}),
)
```

## Endpoints

- `client.ChangePassword(...)` — Change password for the currently authenticated user
- `client.ForgotPassword(...)` — Request a password reset token (always returns 200 to prevent enumeration)
- `client.Health(...)` — Health check endpoint
- `client.Jwks(...)` — Public JSON Web Key Set for verifying access tokens
- `client.Login(...)` — Authenticate a user and receive access + refresh tokens
- `client.Logout(...)` — Revoke the current refresh token family and clear cookies
- `client.LogoutAll(...)` — Revoke all refresh token families for a user
- `client.Refresh(...)` — Exchange a refresh token for new access + refresh tokens
- `client.ResetPassword(...)` — Reset password using a single-use reset token (revoked after use)
- `client.Session(...)` — Validate current session and return user info
- `client.Sessions(...)` — List active sessions (refresh token families) for the authenticated user
- `client.Signup(...)` — Register a new user and receive access + refresh tokens

## Error Handling

All API errors are returned as `*shroudb_auth.ShroudbAuthError` with `Code` and `Message` fields.

```go
_, err := client.Login(ctx, "alice", "wrong-password")
if shroudb_auth.IsUnauthorized(err) {
    fmt.Println("bad credentials")
}
```
