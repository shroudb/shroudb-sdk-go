# Shroudb Go Client

Go client for the [Shroudb](https://github.com/shroudb/shroudb) Credential management server.

## Install

```bash
go get github.com/shroudb/shroudb-go
```

## Quick Start

```go
package main

import (
    "fmt"
    "log"

    shroudb "github.com/shroudb/shroudb-go"
)

func main() {
    client, err := shroudb.Connect("shroudb://localhost")
    if err != nil {
        log.Fatal(err)
    }
    defer client.Close()

    // Issue a credential
    result, err := client.Issue("my-keyspace", &shroudb.IssueOptions{Ttl: 3600})
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(result.CredentialId, result.Token)

    // Verify it
    verified, err := client.Verify("my-keyspace", result.Token, nil)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(verified.State) // "active"
}
```

## Connection URI

```
shroudb://[token@]host[:port][/keyspace]
shroudb+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb://localhost` — plain TCP, default port 6399
- `shroudb+tls://prod.example.com` — TLS
- `shroudb://mytoken@localhost:6399/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```go
client, err := shroudb.Connect("shroudb://localhost", shroudb.PoolConfig{
    MaxIdle: 8,
    MaxOpen: 32,
})
```

## Commands

- `client.Auth(...)` — Authenticate the current connection
- `client.ConfigGet(...)` — Retrieve a runtime configuration value
- `client.ConfigSet(...)` — Set a runtime configuration value
- `client.Health(...)` — Check server or keyspace health
- `client.Inspect(...)` — Retrieve full details about a credential
- `client.Issue(...)` — Issue a new credential in the given keyspace
- `client.Jwks(...)` — Return the JSON Web Key Set for a JWT keyspace
- `client.Keys(...)` — List credential IDs with optional filtering and pagination
- `client.Keystate(...)` — Show the current key ring state for a keyspace
- `client.PasswordChange(...)` — Change a user's password (requires old password)
- `client.PasswordImport(...)` — Import a pre-hashed password for migration from another system (argon2, bcrypt, scrypt)
- `client.PasswordSet(...)` — Set a password for a user in a password keyspace
- `client.PasswordVerify(...)` — Verify a user's password
- `client.Refresh(...)` — Exchange a refresh token for a new one
- `client.Revoke(...)` — Revoke a credential by ID
- `client.RevokeBulk(...)` — Bulk-revoke multiple credentials
- `client.RevokeFamily(...)` — Revoke all credentials in a refresh token family
- `client.Rotate(...)` — Trigger signing key rotation for a keyspace
- `client.Schema(...)` — Display the metadata schema for a keyspace
- `client.Suspend(...)` — Temporarily suspend a credential
- `client.Unsuspend(...)` — Reactivate a previously suspended credential
- `client.Update(...)` — Update metadata on an existing credential
- `client.Verify(...)` — Verify a credential (JWT, API key, or HMAC signature)


## Streaming Subscribe

Subscribe to real-time events on a channel:

```go
sub, err := client.Subscribe("my-channel")
if err != nil {
    log.Fatal(err)
}
defer sub.Close()

for evt := range sub.Events() {
    fmt.Printf("[%s] %s %s at %d\n", evt.EventType, evt.Keyspace, evt.Detail, evt.Timestamp)
}
// Check for read errors after the channel closes
if err := <-sub.Err(); err != nil {
    log.Println("subscription error:", err)
}
```

Each `SubscriptionEvent` contains:
- `EventType` — the type of event (e.g. "issued", "revoked")
- `Keyspace` — the affected keyspace
- `Detail` — additional event detail
- `Timestamp` — Unix timestamp of the event
