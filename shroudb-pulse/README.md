# ShroudbPulse Go Client

Go client for the [ShroudbPulse](https://github.com/shroudb/shroudb-pulse) Observability plane for unified audit event streaming.

## Install

```bash
go get github.com/shroudb/shroudb-pulse-go
```

## Quick Start

```go
package main

import (
    "fmt"
    "log"

    shroudb_pulse "github.com/shroudb/shroudb-pulse-go"
)

func main() {
    client, err := shroudb_pulse.Connect("shroudb-pulse://localhost")
    if err != nil {
        log.Fatal(err)
    }
    defer client.Close()

    // Issue a credential
    result, err := client.Issue("my-keyspace", &shroudb_pulse.IssueOptions{Ttl: 3600})
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(result.CredentialId, result.Token)

    // Verify it
    verified, err := client.Verify("my-keyspace", result.Token, nil)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println(verified.State) // "active"
}
```

## Connection URI

```
shroudb-pulse://[token@]host[:port][/keyspace]
shroudb-pulse+tls://[token@]host[:port][/keyspace]
```

Examples:
- `shroudb-pulse://localhost` — plain TCP, default port 7099
- `shroudb-pulse+tls://prod.example.com` — TLS
- `shroudb-pulse://mytoken@localhost:7099/sessions` — auth + keyspace

## Connection Pool

The client maintains an internal connection pool. Tune it at connect time:

```go
client, err := shroudb_pulse.Connect("shroudb-pulse://localhost", shroudb_pulse.PoolConfig{
    MaxIdle: 8,
    MaxOpen: 32,
})
```

## Commands

- `client.Actors(...)` — Most active actors
- `client.Auth(...)` — Authenticate the connection
- `client.Count(...)` — Count events matching filter arguments
- `client.Errors(...)` — Per-operation error rates
- `client.Health(...)` — Check server health
- `client.Hotspots(...)` — Find hotspot resources with highest activity
- `client.Ingest(...)` — Ingest a single audit event
- `client.IngestBatch(...)` — Ingest a batch of audit events
- `client.Query(...)` — Query events with filter arguments
- `client.SourceList(...)` — List configured event sources
- `client.SourceStatus(...)` — Show per-source ingestion statistics


## Streaming Subscribe

Subscribe to real-time events on a channel:

```go
sub, err := client.Subscribe("my-channel")
if err != nil {
    log.Fatal(err)
}
defer sub.Close()

for evt := range sub.Events() {
    fmt.Printf("[%s] %s %s at %d\n", evt.EventType, evt.Keyspace, evt.Detail, evt.Timestamp)
}
// Check for read errors after the channel closes
if err := <-sub.Err(); err != nil {
    log.Println("subscription error:", err)
}
```

Each `SubscriptionEvent` contains:
- `EventType` — the type of event (e.g. "issued", "revoked")
- `Keyspace` — the affected keyspace
- `Detail` — additional event detail
- `Timestamp` — Unix timestamp of the event
